## 1. 사이트 개요

### 1.1 콘셉트
- **아티스트**: ThevertMenthe (볼펜 드로잉 아티스트)
- **주제**: 심플하고 섬세한 감성의 드로잉 갤러리
- **스타일**: 미니멀 아트 포트폴리오

### 1.2 기술 스택 (추정)
| 기술 | 용도 |
|------|------|
| Three.js | 3D WebGL 렌더링 |
| GSAP | 애니메이션 |
| WebGL | 3D 그래픽 |


## 3. 인터랙션 패턴

### 3.1 스크롤 동작
```
[시작] ─────────────────────────────────────────────────── [끝]
   │                                                         │
   │   부드러운 관성 스크롤 (Lenis/Locomotive 스타일)        │
   │   이미지가 패럴랙스 효과로 살짝 움직임                   │
   │   텍스트 페이드 인/아웃                                 │
   │                                                         │
```

### 3.2 호버 효과
```
[기본 상태]              [호버 상태]
┌─────────────┐          ┌─────────────┐
│             │          │  ╔═══════╗  │
│   이미지    │   ──→    │  ║ 확대  ║  │  + 그림자
│             │          │  ╚═══════╝  │  + 커서 변경
└─────────────┘          └─────────────┘
```

### 3.3 커서 인터랙션
- 기본: 작은 원형 커서
- 호버: 확대 + "VIEW" 텍스트
- 클릭: 축소 피드백

## 5. 권장 사항

| 항목 | ThevertMenthe | 현재 GGP | 권장 |
|------|---------------|----------|------|
| 배경 | 밝은 크림색 | 거의 검정 | **밝은 색으로 변경** |
| 이미지 | 실제 드로잉 | SVG 더미 | **실제 이미지 사용** |
